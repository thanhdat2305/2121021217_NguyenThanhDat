1. CẤU TRÚC DỰ ÁN .NET MVC
- .NET MVC là một framework phát triển ứng dụng web mã nguồn mở của Microsoft, giúp xây dựng ứng dụng bằng cách tách biệt logic ứng dụng thành 3 phần
+ model (dữ liệu)
+ view (giao diện)
+ controller (điều khiển)

nhằm tăng tính linh hoạt, dễ bảo trì và mở rộng, hiện tại nó đã được hợp nhất vào ASP.NET CoreMVC
* user gửi yêu cầu: người dùng truy cập một ULR trên trình duyệt
* controller xử lý: controller nhận yêu cầu xử lý logic nghiệp vụ
